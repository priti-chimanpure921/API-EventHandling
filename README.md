# 📡 API Event Handling in JavaScript

A beginner-friendly project demonstrating how to fetch data from external APIs using event handling in JavaScript. This repo includes mini projects that make API calls with button clicks and display results dynamically using the DOM.

# 🧪 Projects Included

<h3>1. 🐱 Cat Facts & 🐶 Dog Images</h3>

Fetches:

A random cat fact from catfact.ninja

A random dog image from dog.ceo

📍Implemented in: app34.js
📌 HTML Structure: Cat & Dog buttons, a list for facts, and an image placeholder.
👉 Click buttons to load data.


<h3>2. 🎓 Search Indian Colleges</h3>

Fetches a list of colleges in India using the Universities API based on user input.

# 🚀 How It Works

Both projects follow this pattern:

Event Listener listens for button clicks.

On click, a function runs to:

Make an API request using axios.

Process the data.

Update the DOM with the results.

# This teaches:

✔ Asynchronous JavaScript with async/await

✔ Handling browser events (addEventListener)

✔ DOM manipulation (createElement, innerText, etc.)

✔ Working with external APIs in real time

📍Implemented in: app37.js
📌 HTML Structure: Input field + Search button + list to show results.
👉 Enter a state/keyword and click Search.

# 📌 Learnings & Skills

By building this project you’ll learn:

How to handle DOM events (like button clicks)

How to make HTTP requests using axios

How to display API results using the DOM

How to handle responses and errors
